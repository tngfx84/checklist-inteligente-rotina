# Visão futura do projeto

Este projeto está sendo desenvolvido de forma incremental.
A visão de longo prazo é evoluí-lo até uma aplicação full-stack em .NET,
mantendo a separação clara entre dados, regras e interface.

Esta visão não define escopo nem cronograma.
Ela existe apenas para registrar a intenção arquitetural
e evitar perda de contexto ao longo do tempo.
