# Regras do sistema

## Ordenação padrão
- período → prioridade → tipo (fixa antes de variável)

## Regras de adaptação
- Período cheio/comprometido → adiar variáveis de baixa prioridade
- Dependência não atendida → status bloqueada
- Imprevisto → inserir tarefa extraordinária e recalcular

## Regra de consistência
- Tarefas fixas não são removidas; no máximo mudam de período/estado
